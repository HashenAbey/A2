Australian Weather Data Across Key Cities

Interactive visualizations exploring rainfall, temperature, and climate patterns across Australia.

Weather Data Map

This map shows weather data across Australia, with circle size indicating maximum temperatures and color indicating temperature differences. Hover over a location to see detailed information.

Total Rainfall by Month and Location

This bar chart allows you to explore the total rainfall by month across various Australian locations. Use the filter to select different locations.

Max Temperature vs Rainfall Scatter Plot

The scatter plot below represents the relationship between maximum temperature and rainfall. You can adjust the filter to focus on certain temperature ranges.

Heatmap of Rainfall

The heatmap below visualizes total rainfall by month and location. The color intensity represents the rainfall amount, with white borders separating each block for clarity.
